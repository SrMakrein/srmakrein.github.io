Arquetipo OS

Sistema operativo modular escrito en Rust. Arquitectura híbrida diseñada para desarrollo rápido (simulación en host) y despliegue nativo (no_std).

Arquitectura

  • arquetipo_core: Núcleo portable (no_std). Contiene la lógica del sistema, VFS y gestión de comandos.
  • arquetipo_host: Entorno de simulación para Windows/Linux/macOS. Ejecuta el núcleo sobre el OS anfitrión.

Funcionalidades Actuales

  • Shell Interactiva: Intérprete de comandos básico.
  • Sistema de Archivos Virtual (VFS): Sistema de archivos en memoria con soporte para:
    • Navegación: cd, ls (con estadísticas).
    • Gestión: mkdir, touch, cp, mv, delete.
  • Multi-idioma: Soporte dinámico para Inglés y Español (lang <en|es>).

Ejecución (Simulación)

Requisitos: Necesitas tener instalado Rust y Cargo.

cargo run -p arquetipo_host

Brand Environment Status

Módulo custom limpio y extensible para Drupal 11. Genera una matriz de información (Marcas × Entornos: QA, UAT, PRE, PRO) permitiendo visualizar la rama y la versión desplegada en cada nodo de una arquitectura multisite.

Características Principales

  • Lectura de estado de despliegue en tiempo real.
  • Soporte nativo para integraciones con repositorios (ej. GitLab) mediante tokens de acceso.
  • Configuración centralizada en settings.local.php para mayor seguridad.

Configuración de Ejemplo (settings.local.php)

$settings['brand_env_status_auth'] = [ 
  'git.tudominio.com' => [ // Host remoto 
    'headers' => [ 
      // Opción 1: cabecera típica de GitLab con token personal 
      'PRIVATE-TOKEN' => 'glpat-XXXXXXXXXXXXXXXXXXXX',
      
      // Opción 2: alternativa con Authorization Bearer
      // 'Authorization' => 'Bearer GLPAT_AQUI_TU_TOKEN',
    ],
    // Tiempo máximo de espera de la petición (segundos)
    'timeout' => 20,
    
    // Validación de certificados SSL/TLS
    'verify' => true,
  ], 
];

LobbiX (Laravel 11 + DDEV)

Un clon de Reddit enfocado en la simplicidad, ofreciendo un entorno totalmente reproducible utilizando DDEV y Laravel 11.

Requisitos

  • Docker Desktop (con WSL2 en Windows, activar integración WSL para la distro).
  • DDEV ≥ 1.23

Arranque Rápido

Sigue estos pasos para levantar el entorno de desarrollo paso a paso:

# 1) Clonar
git clone <URL_REPO> && cd LobbiX

# 2) Iniciar ddev
ddev start

# 3) Instalar dependencias (si no existen)
ddev exec -w /var/www/html/src composer install

# 4) Crear .env y APP_KEY
ddev exec -w /var/www/html/src cp -n .env.example .env
ddev exec -w /var/www/html/src php artisan key:generate

# 5) (Opcional) Sesiones en archivos para dev
# En .env: SESSION_DRIVER=file
ddev exec -w /var/www/html/src mkdir -p storage/framework/sessions

# 6) Migraciones (si procede)
ddev exec -w /var/www/html/src php artisan migrate

App Stock Calendar

Aplicación web B2B orientada a la compra y venta de pedidos al por mayor, diseñada específicamente para optimizar la comunicación y logística entre proveedores y distribuidores.

Desarrollo Colaborativo

Este proyecto se llevó a cabo en conjunto con un equipo multidisciplinar de compañeros, donde gestionamos los requerimientos del cliente, planificamos hitos de entrega y trabajamos coordinados mediante metodologías ágiles y control de versiones (Git).

Funcionalidades Clave

  • Gestión de Calendario ("Stock Calendar"): Sistema de planificación de envíos y reposición en el tiempo basado en las demandas mayoristas recurrentes.
  • Roles y Permisos: Interfaces y flujos adaptados dependiendo si el usuario accede como proveedor (control de stock, catálogos) o como distribuidor (solicitudes de presupuestos, pedidos en masa).
  • Arquitectura Escalable: Diseñada para soportar altos volúmenes de transacciones e inventariado.

Control por Voz - Macro Launcher Pro

Aplicación portable para Windows que permite controlar tu computadora mediante comandos de voz y atajos de teclado personalizables.

Características Principales

  • Reconocimiento de Voz: Funciona en español. Utiliza una palabra de activación (por defecto "asistente") para escuchar comandos solo cuando lo desees.
  • Atajos de Teclado Globales: Cada macro puede tener un atajo asignado para ejecución instantánea sin necesidad de hablar, incluso con la aplicación minimizada.
  • Gestión de Macros: Interfaz gráfica amigable con selector de archivos. Permite ejecutar programas, scripts o comandos del sistema de forma persistente.

Instalación y Uso

Requiere tener Python instalado en el sistema (marcando "Add Python to PATH" durante la instalación). Tras descargar el programa, cuenta con asistentes de inicio:

launcher.bat (Instalación de dependencias e inicio)
iniciar.bat (Lanzamiento directo)

Ejemplos de Macros Avanzados

  • Ejecutar Scripts: Comando: C:\Scripts\backup.bat | Atajo: Ctrl+Shift+B
  • Abrir Carpetas: Comando: explorer.exe C:\Proyectos | Atajo: Ctrl+Alt+P
  • Comandos del Sistema: Comando: shutdown /s /t 60 (apagar la PC).
